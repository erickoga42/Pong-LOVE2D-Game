🚀 Como Executar os Jogos
Para rodar qualquer um dos jogos, você precisa ter o LÖVE instalado no seu computador.

    Instale o LÖVE: Baixe em love2d.org.
    Clone o repositório:
    bash

    git clone https://github.com

    Use o código com cuidado.
    Execute um jogo:
    Arraste a pasta do jogo desejado para cima do executável do love.exe ou use o terminal dentro da pasta do jogo:
    bash

    love .

    Use o código com cuidado.
